# Study_Java

## 목차 (Table of Contents)

### 자바 기본 실습 (Java Basics Practice)

- **P1_2_06_BoolTest.java**: 불리언 데이터 타입 테스트
- **P1_2_07_LocalVariableType.java**: 지역 변수 타입 테스트
